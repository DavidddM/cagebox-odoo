# Odoo Skills Index

Skills reference for Odoo development across versions 17.0, 18.0, and 19.0.
Consult the relevant file before starting work on any Odoo task.

| File | Covers | When to consult |
|------|--------|-----------------|
| `odoo-module-structure.md` | Manifest, directory layout, assets, data files | Creating new modules, adding assets, data file load order |
| `odoo-orm.md` | Models, fields, inheritance, recordset API, decorators | Any model/field work, ORM method usage |
| `odoo-views-ui.md` | View XML, modifiers, inheritance, form/list/kanban | Writing or modifying views, UI layout |
| `odoo-owl-javascript.md` | OWL components, hooks, RPC, registries, services, widgets | Any JS/OWL frontend work |
| `odoo-security.md` | ACLs, record rules, groups, sudo | Configuring security, debugging permissions |
| `odoo-controllers-api.md` | HTTP routes, request helpers, XML-RPC, JSON-2 API | Web endpoints, external integrations |
| `odoo-testing.md` | Test classes, tags, CLI, Form helper, tours | Writing or running tests |
| `odoo-migrations.md` | Migration scripts, openupgradelib, noupdate, failure fixes | Writing pre/post/end migration scripts |
| `odoo-reporting.md` | Report actions, QWeb templates, layouts, wkhtmltopdf | Creating or customizing PDF/HTML reports |
| `odoo-server-config.md` | CLI flags, odoo.conf, addons_path, logging, dev mode | Server setup, debugging, performance tuning |
| `odoo-common-patterns.md` | Mail thread, sequences, actions, domains, translations | Standard implementation patterns |
| `odoo-version-changelog.md` | Breaking changes, deprecations, new features per version | Before migration, when targeting a specific version |
